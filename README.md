# Astromodel Knowledge Base
**Цифровая база знаний Astromodel** – современный инструмент сбора, хранения и систематизации научно-исследовательских, образовательных и просветительских материалов по астрономии, астрофизике, небесной механике, космологии, методам математического моделирования и другим направлениям деятельности коллектива лаборатории астрофизики БФУ им. И. Канта, Астрономического сообщества БФУ им. И. Канта, а также смежных и партнёрских коллективов, структур и организаций.

Цифровая база знаний используется для централизованного хранения, регулярного обновления, удобной интеграции с внешними ресурсами и безопасного дистанционного доступа к справочной, обучающей и научной информации. База знаний обеспечивает непрерывность процесса обработки информации в различных областях деятельности: сведения, полученные в рамках выполнения научно-исследовательских работ, могут быть немедленно использованы для подготовки образовательных и просветительских материалов. Это позволяет добиться бесшовного использования разнообразных данных во всей полноте деятельности коллективов, оперирующих базой знаний, включая как готовые материалы (конспекты, справочники, изображения и анимации), так и технические данные (описания скриптов и программных модулей, файлы данных с условиями и результатами моделирования, конфигурационные файлы и т.д.).

## Интеграция с курсами на платформе Astromodel

Готовые материалы, разработанные и загруженные в цифровую базу знаний, могут быть открыты для всеобщего доступа на платформе Astromodel Courses. Для этого необходимо создать в директории с курсом файл index.mdx и привести архитектуру в соответствие шаблону, описанному в astromodel_courses_doc.

Таким образом, оптимальным способом наполнения цифровой базы знаний является проработка материалов и их описание по шаблону в файле формата .md. Как только материалы будут готовы к публичному размещению на платформе Astromodel Courses, достаточно будет изменить формат файла на .mdx для автоматической трансляции данных на платформу. В перспективе рассматривается возможность создания специализированного скрипта для автоматизации этой операции.

## Использование цифровой базы знаний

Цифровая база знаний Astromodel используется на постоянной основе для текущего документирования любой научно-исследовательской, образовательной и просветительской деятельности коллектива её авторов. По мере развития того или иного направления деятельности, текущая документация оформляется в виде набора конспектов по теоретическим вопросам и практических заданий. Готовые наборы размещаются в соответствующем разделе курсов на платформе Astromodel, где они могут использоваться как в качестве справочной информации в рамках дальнейшей работы коллектива, так и в роли обучающих материалов для подготовки абитуриентов, студентов, аспирантов и профессиональной переподготовки участников проекта.
